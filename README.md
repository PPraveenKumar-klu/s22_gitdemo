# s22_gitdemo - Git Demonstration Repository

Welcome to the Spring 2022 Git Demo Repository! This repository serves as a practical demonstration of Git version control concepts and workflows.

## Overview

This repository contains sample code and examples to help you learn and practice Git operations, including:
- Basic Git commands
- Branching and merging
- Collaboration workflows
- Conflict resolution
- Best practices

## Learning Resources

### Basic Git Commands
- `git init` - Initialize a new Git repository
- `git clone` - Clone an existing repository
- `git add` - Stage changes for commit
- `git commit` - Commit staged changes
- `git push` - Push commits to remote repository
- `git pull` - Fetch and merge changes from remote

### Branching
- `git branch` - List, create, or delete branches
- `git checkout` - Switch branches
- `git merge` - Merge branches
